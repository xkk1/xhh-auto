# 小红狐自动化开发文档

## web API

### 本地文件映射

GET /目录/数据/<文件路径文件名>: 映射本地数据目录文件  
GET /目录/小红狐/<文件路径文件名>: 映射本地小红狐目录文件  
GET /目录/小红狐父/<文件路径文件名>: 映射本地小红狐父目录文件  
GET /目录/脚本/<文件路径文件名>: 映射本地脚本目录文件

### 数据

GET /数据/<文件路径文件名>: 获取本地最新 json 数据  
POST /数据/<文件路径文件名>: 创建或更新 json 数据

GET /数据/?标识=<文件标识>: 获取指定标识的内存 json 数据  
POST /数据/?标识=<文件标识>: 创建或更新指定标识的 json 数据
