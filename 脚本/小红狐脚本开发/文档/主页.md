# 小红狐自动化开发文档

## 数据存放位置推荐



### 全局配置

```python
# 数据/全局配置/<脚本模块名>
路径: pathlib.Path =  小红狐.核心.配置.获取全局配置目录("脚本模块名")
# 数据/全局配置/<脚本模块名>/数据.json
数据: 小红狐.工具.数据工具.数据 = 小红狐.核心.配置.获取配置数据("脚本模块名")
```

## web API

### 本地文件映射

GET /api/目录/数据/<文件路径文件名>: 映射本地数据目录文件  
GET /api/目录/小红狐/<文件路径文件名>: 映射本地小红狐目录文件  
GET /api/目录/小红狐父/<文件路径文件名>: 映射本地小红狐父目录文件  
GET /api/目录/脚本/<文件路径文件名>: 映射本地脚本目录文件

### 数据

GET /api/数据/<文件路径文件名>: 获取本地最新 json 数据  
POST /api/数据/<文件路径文件名>: 创建或更新 json 数据
可选参数 ?保存 ，含有此参数时立即保存数据到文件

GET /api/数据/?标识=<文件标识>: 获取指定标识的内存 json 数据  
POST /api/数据/?标识=<文件标识>: 创建或更新指定标识的 json 数据
