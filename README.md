## 运行环境

1. 安装 Python 3.12
2. 安装 git 克隆仓库 `https://gitee.com/xkk1/letsgo-xhh.git` 或前往 <https://gitee.com/xkk1/letsgo-xhh/repository/archive/main.zip> 下载压缩包
3. 安装依赖包 `pip install -r requirements.txt`
4. 安装 Playwright 浏览器 `playwright install chromium --no-shell` （打包参考 <https://playwright.dev/python/docs/library#pyinstaller>）
5. 运行程序 `python 元梦之星小红狐.py`
